# Mini Educational Compiler

## System Requirements
- Python 3.9+
- No external dependencies

## Setup Instructions
1. Clone repository
2. Navigate to project folder

## Run Instructions
python main.py

## Example Output
25

## Features
- Arithmetic expressions
- Variable assignment
- Print statements
- AST generation
- Symbol table implementation

## Known Limitations
- No floating-point numbers
- No conditionals
- No loops
- No functions
